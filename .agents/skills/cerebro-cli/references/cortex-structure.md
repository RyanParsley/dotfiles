# Cortex Content Structure

## config.toml

```toml
[[projects]]
name = "cerebro"
repo_path = "/Users/ryan/Projects/cerebro"
active = true

[[projects]]
name = "old-project"
repo_path = "/Users/ryan/Projects/old-project"
active = false
```

## Project Pages

### Generated (`content/projects/<name>.md`)

Auto-generated by `cerebro build`. Contains:
- Project metadata (repo path, active status)
- Recent sessions (AI assistant conversations)
- Recent commits
- Embedded TODOs found in source code
- Last activity timestamp

### Manual Notes (`content/notes/projects/<name>.md`)

Hand-written by the developer. Free-form markdown with optional sections:
- `## Status` — current project status
- `## Journal` — recent journal entries
- `## Next` — next steps / action items

## Journal Entries

Path: `content/journal/YYYY/MM/DD.md`

Generated daily by `cerebro build`. Contains sessions and commits for that date.

## Intent Files

Path: `content/intent/<period>/<identifier>.md`

Hand-written by the developer. Free-form markdown describing goals/intentions for the period.
